ElasticSearch Indexing for an Application Classifier

Information Retrieval in Personal Computers, based on Semantics of Query
The search tool developed searches for documents on a personal computer, based on analysis of the query eneterd. The query is first processed using a set of common naming conventions observed. The semantics of the keywords and the usage history is then used to fetch results. Developed using C and C++.
Dailython, an Android Application
This application aims to change daily tasks into challenges. Users can set goals and rewards for themselves and even challenge their friends to complete their tasks. Developed using Android SDK for Eclipse and Google Cloud database.
Luminary, an Android Application
This application fetches an inspirational quote, anecdote or story for the user, at a fixed time everyday. The user can also submit tales that inspire them. Developed using HTML, CSS, JQuery and Polymer and deployed on Android using Adobe PhoneGap.
Bookstore Application
This application streamlines the traffic in a bookstore. It maintains an inventory of all books for the user. Users can check the latest arrivals, submit reviews and even get suggestions from popular websites. The admin of the bookstore has a facility to manage the inventory and monitor the employee database. Developed using Java and local MySQL database.
Scoring Application

Lexical Analyser and Compiler using Sanskrit
This paper proposes the construction of a C-based compiler in Sanskrit language. It demonstrates the construction of a lexical analyser for Sanskrit, that suggests correction to simple Sanskrit sentences. The aim of this paper is to help everyone code, without any language barrier. Developed using C and C++.
